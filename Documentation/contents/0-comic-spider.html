Overview

Project home

The purposes of this software is sharing and learning which means it is for previewing purposes only. If you like the mangas/illustrations please support the author by purchasing them.
It won't have a search function, since the original sites always better for leading users to find their interests. It won't have a push function, since the RSS and Google Reader is really doing their best for tracing user's favorites.
It has a programmable interface to control the search behavior of producer: Lua Interface.
I provided some useful methods such as levenshtein_distance and json_decode to help to find target info more efficiently.
Since it's open source, please feel free to define your own version.

If you have any problem, please visit the Issues Page.

If you want to contact me, please leave a msg to my public mail box.

Features

  • Open source, no ads and free forever.
  • Simple drag and drop is all it needs to download.
  • Online auto login service.
  • A better way to browse manga via auto created view pages.
  • Multithread and fast.
  • Has an easy to use Lua programmable interface.
  • Not only comic images, it can be easily modified into a complex downloader.
  • Support remote script controller which means you can always get the latest parser without updating the program.

System Requirement

  • Microsoft .NET Framework 3.5
  • VC++ 2008 runtime (most PCs have already installed, I put it into the 'ComicSpider/Asset/vcredist_x86.exe')
